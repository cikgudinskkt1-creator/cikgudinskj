# SciVerse AR Premium

Versi premium aplikasi AR pembelajaran Sains Tahun 4.

## Ciri Premium
- Login murid/guru
- Firebase realtime leaderboard
- Sistem lencana
- AR pembelajaran
- AI chatbot
- Responsive Android/iOS/Web

## Publish ke Play Store

1. Install Flutter
2. Run:
   flutter pub get

3. Generate APK:
   flutter build apk --release

4. Upload ke Google Play Console

## Cadangan Integrasi
- Firebase
- Unity AR
- OpenAI API
